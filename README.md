# AstroCal

AstroCal is an Android (Jetpack Compose) astronomy utility app for observers.

## Features (v0.0.1)
- **Finder**: PS1 cutout + Gaia overlay (WIP)
- **Celestial**: Aladin Lite sky viewer (WIP)
- **Weather**: Forecast + cloud map shortcuts
- **Tools**: Angle separation, time conversions
- **Languages**: 简体中文 / 繁體中文 / English / Italiano / 日本語 / 한국어

## Download
Go to **Releases** and download the latest APK.

## Notes
This is an early alpha release. Bugs and UI changes are expected.
